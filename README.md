# Raising Impact

Evidence-led philanthropic advisory website helping donors maximise social impact through rigorous research, cost-effectiveness analysis, and transparent methods across India and Asia.

## Tech Stack

- [Astro](https://astro.build/) — static site framework
- Deployed on [Vercel](https://vercel.com/)

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
  components/   # Header, Footer
  layouts/      # Base Layout
  pages/        # index, about, contact
  styles/       # global.css
public/         # Static assets (images, video)
```

## Pages

- `/` — Home (Hero, Pillars)
- `/about` — About Raising Impact
- `/contact` — Contact page
